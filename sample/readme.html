On the other hand, standalone-sequence.svg is standalone, it does not refer to other SVGs, and as such it is more portable (and should work on more html viewers); it can be used when the above does not work. The visual result, though, is exactly the same:

Both these files are using some class attributes for CSS styling: fresh-stroke, painted-stroke and background-stroke. Look into characters.css and into the SVG files themselves for details.
